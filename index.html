Data Mining Project - Housing Price Prediction

Implemented SARIMA, ARIMA, and additional machine learning models to predict housing prices in Boulder, CO, leveraging detailed data pre-processing, feature engineering, and model evaluation techniques.

Data preprocessing and feature selection. Collected housing price data and key attributes such as property size, neighborhood details, and economic factors. Ensured data cleaning and normalization.

Data preprocessing and feature selection. Collected housing price data and key attributes such as property size, neighborhood details, and economic factors. Ensured data cleaning and normalization.

ACF and PACF analysis. Autocorrelation Function (ACF) and Partial Autocorrelation Function (PACF) were used to determine lag terms and seasonality in the time series. This helped configure ARIMA and SARIMA models effectively.

Residual analysis for model validation. Checked residuals to ensure they were normally distributed and exhibited no significant autocorrelation, indicating a well-fitted model. Performed Ljung-Box tests to validate assumptions.

ARIMA model performance evaluation. Tested various configurations of ARIMA to identify optimal lag, differencing, and seasonality parameters. Explored ACF and PACF plots for parameter tuning.

SARIMA implementation. Incorporated seasonality to capture cyclical trends in Boulder’s housing market. Evaluated performance based on RMSE and MAE metrics, with seasonality accounting for quarterly fluctuations.
